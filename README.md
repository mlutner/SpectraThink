# Spectra

**A gymnasium for the mind.**

Spectra is a cognitive development platform that uses AI to challenge your thinking, surface hidden assumptions, and track how your reasoning develops over time.

> "Train the skill AI can't replace."

## Architecture

| Layer | Stack |
|-------|-------|
| Frontend | React (Vite) · TypeScript · Motion · Tiptap · Sigma.js |
| Backend | Node.js/Express (CRUD) · FastAPI (AI ops) |
| AI | Anthropic API · Deepgram (Voice) · pgvector |
| Database | PostgreSQL · Redis |
| Infra | Vercel · Railway · Clerk |
